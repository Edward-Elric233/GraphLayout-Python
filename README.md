本仓库用于记录中南大学可视化实验室网络黑产小组王江奎的工作进度

> 输入文件样式：

```
u1 v1 w1
u2 v2 w2
...
un vn wn
```

其中`ui　vi`为边的节点，`w`为权重，点的编号应该是连续的，如果不是需要进行离散化处理

> 输出文件样式：

**input.csv**

```
source,target,
u1,v1,
u2,v2,
...
un,vn,
```

其中`ui vi`为处理后的节点

**ouput.csv**

```
id,x,y,k,
1,x1,y1,k1,
2,x2,y2,k2,
...
n,xn,yn,kn,
```

`id`为点的编号，`x y`为经过算法处理以后点的坐标，`k`为点的核数
